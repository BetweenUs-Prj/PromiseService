📋 테스트 사전 준비사항

  1. 카카오 로그인: UserService (8081)에서 카카오 로그인 완료
  2. talk_message 권한: 로그인 시 메시지 발송 권한에 동의
  3. 액세스 토큰 확보: 브라우저 개발자 도구에서 토큰 복사
  4. 서버 실행: PromiseService (8080) 정상 실행 중
⚠️ 주의사항: 실제 카카오톡이 발송되므로 테스트용 계정을 사용하세요.
