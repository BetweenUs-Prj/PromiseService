🧪 시드 테스트 스크립트

⚠️ 중요: 아래 스크립트는 브라우저 콘솔에서만 실행하세요!
(F12 → Console 탭에 붙여넣기)

📋 실행 순서

  1. PromiseService(8080) 실행 확인
  2. UserService(8081)에서 카카오 로그인 (JWT 토큰 획득)
  3. 브라우저 콘솔에 아래 스크립트 붙여넣기

🚀 즉시 카카오톡 발송 스크립트 (NEW!)

(async () => { const t = new URLSearchParams(location.search).get('token') || localStorage.getItem('jwt'); if (!t) return console.error('JWT 없음'); localStorage.setItem('jwt', t); const BASE = 'http://localhost:8080'; const r = await fetch(BASE + '/api/dev/seed-and-notify-me', { method:'POST', headers:{Authorization:'Bearer '+t} }); const result = await r.text(); console.log('📱 즉시 카톡 발송 결과:', result); })();

🛠 기존 단계별 테스트 스크립트

(async () => { const qs = new URLSearchParams(location.search); const token = qs.get('token') || localStorage.getItem('jwt'); if (!token) { console.error('JWT 없음: 먼저 카카오 로그인'); return; } localStorage.setItem('jwt', token); const BASE = 'http://localhost:8080'; const headers = { Authorization: 'Bearer ' + token }; const call = async (p, opt={}) => { const r = await fetch(BASE + p, { ...opt, headers:{...headers, ...(opt.headers||{})} }); const t = await r.text(); let j; try{ j=JSON.parse(t);}catch{} console.log(`[${r.status}] ${p}`, j ?? t); return {r,j,t}; }; // 0) 서버 연결 확인 try { const ping = await fetch(BASE + '/ping'); console.log('🏓 서버 연결:', ping.status); } catch (e) { console.error('❌ 8080 연결 불가', e); return; } // 1) 약속 시드 생성 console.log('🌱 시드 약속 생성 중...'); const seeded = await call('/api/dev/seed-appointment', { method:'POST' }); if (!seeded.j?.success) return; // 2) 생성된 약속 확정 const id = seeded.j.appointmentId; console.log(`📋 약속 확정 중... (ID: ${id})`); await call(`/api/appointments/${id}/confirm`, { method:'POST' }); console.log('🎉 테스트 완료! 카카오톡을 확인해보세요!'); })();

📊 기대 결과

  • [200] /api/dev/seed-appointment {success: true, appointmentId: 1}
  • [200] /api/appointments/1/confirm {success: true, ...}
  • 카카오톡 알림 발송! 📱

🔍 체크포인트

  • 자바 파일은 서버 소스에, JS는 브라우저 콘솔에
  • PromiseService(8080)가 실행 중이어야 함
  • UserService(8081)와 JWT 시크릿이 동일해야 함
  • 컨텍스트 경로가 있으면 BASE URL 수정
