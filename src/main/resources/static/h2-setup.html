4단계: 데이터 확인

📊 삽입된 데이터 조회

-- 약속 목록 확인 SELECT id, title, place, start_at, status, host_user_id FROM appointments ORDER BY id; -- 참여자 현황 확인 SELECT ap.appointment_id, a.title, ap.user_id, ap.kakao_id, ap.state, ap.notify_status, ap.notify_error FROM appointment_participants ap JOIN appointments a ON ap.appointment_id = a.id ORDER BY ap.appointment_id, ap.id; -- 카카오 토큰 현황 확인 SELECT kakao_id, SUBSTRING(access_token, 1, 20) || '...' as access_token_preview, access_token_expires_at, talk_message_agreed FROM kakao_auth ORDER BY kakao_id;
✅ 데이터 설정 완료!
이제 다음 API들을 테스트할 수 있습니다:
  • POST /api/appointments/1/confirm - 약속 확정
  • GET /api/appointments/1/participants - 참여자 조회
  • POST /api/appointments/1/notify/resend - 알림 재발송
  • POST /api/appointments/3/cancel - 약속 취소
